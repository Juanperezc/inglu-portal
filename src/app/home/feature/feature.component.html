Características de INGLU

  • Portal web

    Es un espacio de carácter informativo en su mayoría, donde se dará a conocer el club, así como su funcionamiento, incluyendo como registrarse en el club, considerando toda información de interés para miembros y no miembros del mismo

  • Aplicación web desktop

    Es de uso exclusivo para el personal del club, los usuarios podrán acceder a través de un navegador, el mismo contara con un modulo que tiene un sistema de autenticación y estará restringido según el rol que desempeñen los usuarios, este le mostrará información pertinente de acuerdo al usuario.

  • Aplicación móvil

    Compatible con dispositivos que posean sistema operativo Android, dicha aplicación es de uso exclusivo para los pacientes
